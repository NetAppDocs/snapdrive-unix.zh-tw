= 部署工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在您建立Snapshot複本搭配SnapDrive 使用效益技術之前、您必須先安裝Linux主機公用程式（如果使用FC或iSCSI傳輸協定）、安裝SnapDrive 適用於UNIX的版本、檢查支援SnapDrive 的組態以供使用、配置儲存設備、然後設定從Linux主機連線至NetApp儲存系統的連線。
