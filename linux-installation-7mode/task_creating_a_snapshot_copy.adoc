= 建立Snapshot複本
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用單一命令、在網路附加儲存設備（NAS）中使用儲存區域網路（SAN）和NFS磁碟區、來建立多個檔案系統、主機磁碟區、LUN或磁碟群組的Snapshot複本。

.步驟
. 建立Snapshot複本：
+
fs-fs_file_spec_ SnapDrive -snapname _snapsnapshot複製名稱_*
