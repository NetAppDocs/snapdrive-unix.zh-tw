= 由適用於UNIX的文件SnapDrive
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
適用於UNIX的支援安裝多個檔案、例如執行檔、組態檔、解除安裝檔案、診斷檔案和手冊頁、SnapDrive 在主機上。透過這些檔案、您可以偵測問題、將產品目前版本的組態資訊等資訊儲存在SnapDrive 適用於UNIX的支援程式庫中。
