= 用於從主機中斷開在LUN上建立的檔案系統的命令語法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要使用「SnapDrive 停止儲存設備連線」命令移除直接在LUN上建立的檔案系統、請使用下列語法：

「SnapDrive 停止儲存- FS file_spec [-fstype][-vmtype類型]」
