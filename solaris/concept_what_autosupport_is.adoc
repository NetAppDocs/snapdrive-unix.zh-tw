= 什麼是呢AutoSupport
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
支援UNIX的支援功能、可將因執行故障而產生的任何故障、傳送至儲存系統的EMS記錄檢視（位於「/etc/log/EMS」檔案中）AutoSupport SnapDrive SnapDrive 。
