= 準備部署
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
安裝SnapDrive 適用於UNIX的主機必須符合特定的軟體、硬體、瀏覽器、資料庫和作業系統需求。如需有關需求的詳細資訊、請參閱互通性對照表工具。

. 確認您擁有所需的授權。
. 驗SnapDrive 證您的組態是否支援此功能。
. 驗證您的Linux主機是否符合SnapDrive 需求。
